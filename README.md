# Stores
Aplicacion que permite registrar tiendas, guardarlas o eliminarlas. Tambien podemos configurarla como favoritos,llamar a dicha tienda o dirigirse a su pagina principal

En esta aplicacion he utilizado:
✔️Recyclerview
✔️Room
✔️Glide
✔️Fragment
✔️Anko
✔️
